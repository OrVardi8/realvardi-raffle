# 🦄 הגרלת חדי-הקרן · RealVardi

אתר הגרלות עצמאי שרץ במלואו בדפדפן — אין צורך בשרת. הזוכה נבחר בהגרלה הוגנת (Fisher-Yates עם `crypto`), והמירוץ הוא רק ההצגה החזותית של התוצאה.

## מה יש בתיקייה
- `index.html` — האתר עצמו (קובץ יחיד, כולל עיצוב וקוד).
- `logo.png` — הלוגו של RealVardi.
- `participants-sample.csv` — קובץ דוגמה לבדיקה.

## הפעלה מקומית (לבדיקה)
פשוט פותחים את `index.html` בדפדפן (דאבל-קליק). זהו.

## פריסה ל-Netlify

### דרך 1 — גרירה ידנית (הכי מהיר)
1. היכנס ל-https://app.netlify.com → **Add new site → Deploy manually**.
2. גרור את **כל תיקיית `site`** (לא רק הקובץ) לאזור ההעלאה.
3. תוך שניות מקבלים כתובת חיה (לדוגמה `random-name.netlify.app`).

### דרך 2 — דרך Git (לעדכונים אוטומטיים)
1. העלה את התיקייה ל-GitHub.
2. ב-Netlify: **Add new site → Import from Git** ובחר את הריפו.
3. **Build command:** השאר ריק. **Publish directory:** `site` (או `/` אם הקובץ בשורש).

## חיבור הדומיין שלך
1. ב-Netlify: **Site configuration → Domain management → Add a domain**.
2. הזן את הדומיין שברשותך.
3. אצל ספק הדומיין הוסף את הרשומות ש-Netlify נותן:
   - דומיין ראשי (`example.com`) → רשומת **A** לכתובת `75.2.60.5`.
   - תת-דומיין (`www`) → רשומת **CNAME** ל-`<your-site>.netlify.app`.
4. Netlify מנפיק תעודת SSL (HTTPS) אוטומטית תוך כמה דקות.

## על Supabase
הגרסה הזו לא צריכה Supabase — הכול רץ בדפדפן. אם בעתיד תרצה לשמור היסטוריית הגרלות / רשימות משתתפים בענן, אפשר להוסיף את זה. רק תגיד.

## איך משתמשים באתר
1. טוען קובץ CSV (שם בכל שורה, או מופרדים בפסיקים) **או** מקליד שמות ידנית.
2. בוחר כמה זוכים רוצים.
3. לוחץ **התחל את המירוץ** — חדי-הקרן רצים, והמגיע ראשון מנצח.
4. אפשר להריץ הגרלה חוזרת עם אותם משתתפים, או להתחיל רשימה חדשה.
